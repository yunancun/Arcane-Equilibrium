**規則：E2 審查和 E4 回歸是強制步驟，任何情況不得跳過。**

**Why:** 用戶明確要求所有任務按照 §十三 強制工作鏈執行。即使是 P0 緊急通道（快速通道），也必須包含 E2+E4。

**How to apply:**
- E1/E1a 完成後 → 自動派發 E2 審查所有改動的生產代碼文件
- E2 通過（或 CONDITIONAL PASS）後 → 派發 E4 全量回歸
- E4 通過後 → PM 確認本 Wave 完成
- 新增的測試文件（E4 產出）不需要 E2，因為不改生產代碼
- 每個 Wave 都必須走完：E1/E1a → E2 → E4 → PM 確認 → 下一個 Wave

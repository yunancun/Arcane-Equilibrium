**定性**：FinceptTerminal = Bloomberg-style 數據研究終端（C++20/Qt6 + 嵌入式 Python），**不是自動執行引擎**。它的「成熟」在數據廣度 + pip 庫整合 + LLM 人格，**不在原創量化/執行邏輯**。對我們最痛的 crypto-perp alpha 發現，回測/量化/執行/組合風險四層幾乎 0 幫助（比我們弱、或只是 pip 套殼、或股票日線導向）。

**硬約束**：License = **AGPL-3.0**（雙授權）。OpenClaw 專有網路服務 → **一行源碼都不能抄**（會逼整系統開源）。借鑒只能是「思路 + 它封裝了哪個 pip 庫→我們直接裝」。

**值得拿的 3 類（按價值排序）**：
- **Tier 1①另類數據信號源 = 唯一直擊 alpha 痛點的發現**：Polymarket 免費 REST+WS（`gamma-api`/`clob`/`data-api` + `wss://ws-subscriptions-clob.polymarket.com/ws/market`，`PolymarketService.h:11-97`；認證下單 pip `py-clob-client`）+ Kalshi 免費（`api.elections.kalshi.com/trade-api/v2` 宏觀事件機率，`KalshiRestClient.h:44-101`）。**非價格、非 beta 的獨立信號軸**，攻「5 候選全死於 down-market beta 偽裝 edge」根因。
- **Tier 1②`finagent_core/agentic/` agent 學習基礎設施（repo 唯一原創代碼，思路非源碼）**：Reflexion loop（`reflector.py` 3 層 critic：正則→heuristic 跳過→帶預算 LLM critic）/ ReflexionStore（`reflexion_store.py` SQLite FTS5 跨任務教訓庫）/ BudgetGuard（`budget.py` token+USD+wall-clock+step 四維封頂）/ Skill Library（`skill_library.py` Voyager 式成功流程食譜）。底層 `agno` 框架（我們用 Claude Code subagent 不需換）。
- **Tier 2**：MCP ToolDef 模式把我們 funding/OI/liq/CVD/orderbook 包成結構化 tool 給 L2（`McpService`+`CryptoTradingTools.cpp:10-163`，和我們架構契合）/ `river` 線上 drift（`qlib_online_learning.py:176` ADWIN）/ `skfolio`（GerberCovariance 對 crypto 高峰度、DistributionallyRobustCVaR Wasserstein 魯棒，直接 pip）/ DataHub Producer pub/sub TTL+rate-limit 模式 / SecretRedemptionServer Unix-socket 單次贖回密鑰 / Node Editor 真 DAG 執行器（`WorkflowExecutor.cpp:265` 拓撲排序+型別 port+gate_or_queue 半自動核准）作 operator 面向可視化策略編排 UI。
- **Tier 3 思路**：Avellaneda-Stoikov 報價+PIN 毒性流（`qlib_high_frequency.py:108-210` 少數原創）/ SB3+自訂 Gym env 做 exit-policy RL（攻 cut-winners/let-losers）/ GEX-IV surface（`options/gex_calculator.py`，需 Deribit 才對 crypto 有意義）。

**明確忽略**：回測全層（vectorbt/backtesting.py/bt/fast_trade/zipline 套殼，54 檔零 funding/perp，DSR 比我們簡單無 PBO/CSCV）/「Factor Discovery」是行銷（`qlib_service.py:30-37` 直接 `QLIB_AVAILABLE=False`）/ 股票投資人人格（Buffett 等 11 個）/ hedgeFundAgents（`signal_discovery.py:246-248` 量化結果寫死假值沒接進 LLM）/ corporateFinance DCF / geopolitics-maritime / 印度券商 / LLM 路由 / 新聞情緒（關鍵詞計數非 NLP）。**它的「AI 交易」本質=讓多 LLM 看 RSI/MACD 比賽下單（Alpha Arena C++ + agno_trading），我們記憶已證實不產統計 edge。**

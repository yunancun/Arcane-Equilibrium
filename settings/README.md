# settings

这里存放交易 AI 系统的配置文件、规则文件和说明文档。
This folder stores configuration files, rule files, and notes for the trading AI system.

主要子目录说明：
Main subfolder overview:

- environment_files：环境变量和运行参数文件。
- environment_files: environment variables and runtime parameter files.

- secret_files：密钥、API 凭据和其他敏感信息文件。
- secret_files: key files, API credentials, and other sensitive information files.

- trading_rules：交易策略规则和交易行为参数。
- trading_rules: trading strategy rules and trading behavior parameters.

- risk_control_rules：风控限制和保护规则。
- risk_control_rules: risk control limits and protection rules.

- asset_role_rules：资产角色规则，例如结算资产、平台手续费币、基础资产和报价资产的定义。
- asset_role_rules: asset role rules, such as settlement assets, platform fee tokens, base assets, and quote assets.

- ai_prompt_templates：AI Agent 使用的提示词模板。
- ai_prompt_templates: prompt templates used by AI agents.

- system_notes：写给人类看的系统说明和配置备注。
- system_notes: human-readable system notes and setup explanations.

这个目录只放配置和规则，不放真实业务数据。
This folder should contain configuration and rules only, not actual business data.

# binance

这里存放 Binance 相关敏感凭据。
This folder stores sensitive credentials related to Binance.

目录结构分为：
The folder structure is divided into:

- read_only：只读权限凭据
- read_only: credentials with read-only permissions

- demo：测试或演练环境凭据
- demo: credentials for testing or demo environments

- live：实盘环境凭据
- live: credentials for live trading environments

每个子目录下应分开保存：
Each subfolder should store separately:

- api_key
- api_key

- api_secret
- api_secret

Binance 当前在本系统中主要作为辅助交易所和扩展预留。
Binance is currently used in this system mainly as a secondary exchange and reserved expansion path.

🧾 OMS / 执行状态 SM-03

订单生命周期和执行边界
SM-03 管理订单从创建、校验、提交、成交到取消或失败的生命周期。只有通过授权、租约和风控的意图才应进入此状态机。
了解更多 / Learn more
它回答“这笔交易是否已经成为订单、订单是否已提交、最终是否成交或终止”。SM-03 不授予交易权限；它执行前必须依赖 SM-01、SM-02 和 SM-04 的结果。

🛡 Risk Governor / 风控治理 SM-04

风险等级和模式
SM-04 根据账户、持仓、回撤和异常信号调整风险等级。风险升高时自动收缩；风险降低通常需要明确审批。
了解更多 / Learn more

🐣 Graduated Canary Cohort Status / 漸進金絲雀 Cohort 狀態 AMD-03

Demo 演示 未连接 / Not Connected

Demo 演示 需要配置 API 密钥才能使用。请在环境配置中设置 Bybit Demo API 密钥。

如何配置

1. 在 demo-trading.bybit.com 注册并获取 API 密钥

2. 将密钥添加到环境配置文件 .env：

BYBIT_DEMO_API_KEY=your_key_here
BYBIT_DEMO_API_SECRET=your_secret_here

3. 重启交易系统以加载新配置

Demo 演示 使用真实市场数据但虚拟资金，订单在 Bybit 的沙盒服务器上执行。与本地 Paper 模拟的区别是：Demo 订单会经过 Bybit 的撮合引擎，更接近真实环境。

❌ 關閉所有 Demo 倉位

此操作將通過 Bybit Demo API 市價平掉所有 Demo 帳戶倉位並取消掛單。
使用虛擬資金，不影響真實帳戶。

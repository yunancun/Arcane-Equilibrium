Bybit Demo 未连接 / Not Connected

Bybit Demo 交易需要配置 API 密钥才能使用。请在环境配置中设置 Bybit Demo API 密钥。

如何配置 / How to configure

1. 在 demo-trading.bybit.com 注册并获取 API 密钥

2. 将密钥添加到环境配置文件 .env：

BYBIT_DEMO_API_KEY=your_key_here
BYBIT_DEMO_API_SECRET=your_secret_here

3. 重启交易系统以加载新配置

Bybit Demo 使用真实市场数据但虚拟资金，订单在 Bybit 的沙盒服务器上执行。这与本地 Paper Engine 的区别是：Demo 订单会经过 Bybit 的撮合引擎，更接近真实环境。

⚠ 計劃重啟服務器

風險說明 / Risks
• 重啟期間（約 10-30 秒）市場監控中斷，無法執行止損
• 未平倉的 Paper 模擬 倉位在重啟後恢復，但可能錯過止損時機
• Bybit Demo 演示 倉位不受影響（由交易所端管理）
• 重啟後系統自動恢復策略和市場數據訂閱
選擇重啟延遲時間 / Select restart delay

強制清倉？/ Force Liquidate?

清倉規則 / Liquidation Rules
系統會立即平倉所有扣除手續費後仍為盈利的 Paper 模擬 倉位。
虧損倉位不會被強制平倉，以避免鎖定損失。
The system will close all Paper positions where net PnL after fees > 0.
Losing positions will not be closed.

▶ 启动双引擎 / Start Engines

新一轮测试将归零所有指标（历史数据保留供分析），启动后 30 秒预热期间不会执行交易。选择「从头开始」将同步 Bybit Demo 余额作为初始资金。

A new round resets all metrics (historical data preserved). 30s warmup blocks trading after start. "Fresh start" syncs Demo balance as initial capital.

⚠ Bybit Demo 如需重置余额，请到 demo-trading.bybit.com 手动操作。

⏹ 停止双引擎 / Stop Engines

停止将自动平掉所有 Paper 和 Bybit Demo 持仓，取消所有挂单和条件单，并结算本轮 PnL。历史数据会保留供分析。

Stopping will auto-close all Paper & Demo positions, cancel all orders, and finalize PnL. Historical data is preserved for analysis.

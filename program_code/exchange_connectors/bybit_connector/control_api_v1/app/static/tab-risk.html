📊 仓位控制与曝险 / Position Sizing & Exposure

Per-Trade Risk Budget / 单笔风险预算
每笔交易投入的最大余额比例（可调软参数，非 P1 硬限制）。越大允许单笔越重，越小则单笔更轻。
关键影响：BTC/ETH 等高价资产需要至少 5-10% 才能产生最小手数
%
Max Single Position / 单笔最大持仓
单个仓位占账户余额的最大比例。防止单一仓位过于集中
过高 → 单笔亏损可能致命 | 过低 → 高价资产无法开仓
%
Max Total Exposure / 总曝险上限
所有持仓市值总和占账户余额的最大比例。超过则拒绝新开仓
100% = 满仓上限 | 200% = 允许 2x 杠杆曝险
%
Max Correlated Exposure / 关联曝险上限
高度相关持仓的总曝险上限（空 = 不限制）。用于控制关联品种集中风险
过低 → 限制对冲策略 | 过高 → 关联风险失控
%
Allowed Categories / 允许品类
允许交易的品类，逗号分隔（例：linear,spot）。Rust 引擎直接过滤
留空 = 不过滤 | linear = 仅永续合约
Max Same-Direction / 同向最大持仓数
同一方向（全做多/全做空）的最大持仓数量。Guardian 超限则拒绝
过高 → 方向性集中风险 | 过低 → 限制策略多样性

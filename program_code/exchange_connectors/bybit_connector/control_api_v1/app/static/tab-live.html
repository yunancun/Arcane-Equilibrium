⏹ 停止 Live 实盘 引擎

停止將執行以下操作：
1. 取消所有 Live 实盘 掛單
2. 平掉所有 Live 实盘 持倉（市價）
3. 引擎進入觀察模式（observing），停止新訂單
⚠ 平倉操作使用真實資金，請確認市場流動性充足。

🚨 緊急停止 / Emergency Stop

緊急停止將立即：
1. 強制取消所有掛單
2. 強制市價平倉所有持倉
3. 封鎖引擎 — 不接受任何新指令直到手動解除
⚠ 高滑點風險。僅在緊急情況下使用。

⚠ 此操作無法撤銷，所有平倉單立即市價成交。

❌ 關閉所有 Live 实盘 倉位

此操作將通過 Bybit API 市價平掉所有 Live 实盘 持倉並取消掛單。
Session 不會停止 — 引擎繼續運行，策略可繼續開新倉。
⚠ 市價成交，存在滑點風險。真實資金。

# bybit_connector

这里存放 Bybit 交易所连接器相关代码。
This folder stores code related to the Bybit exchange connector.

后续主要会包括：
This folder will later mainly include:

- REST API 调用代码
- REST API client code

- WebSocket 订阅与事件处理代码
- WebSocket subscription and event handling code

- 账户、订单、成交、仓位同步逻辑
- account, order, fill, and position synchronization logic

- 只读模式、测试模式和实盘模式切换逻辑
- logic for switching between read-only, test/demo, and live modes

当前阶段先用于搭建骨架，不直接执行实盘交易。
At the current stage, this folder is used to build the connector skeleton and does not directly execute live trading.

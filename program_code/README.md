# program_code

这里存放交易 AI 系统的程序代码。
This folder stores the source code of the trading AI system.

主要子目录说明：
Main subfolder overview:

- market_data_collector：用于抓取交易所行情、新闻或其他外部数据的代码。
- market_data_collector: code used to collect exchange market data, news, or other external data.

- market_data_processor：用于清洗、整理和加工市场数据的代码。
- market_data_processor: code used to clean, transform, and prepare market data.

- trading_strategy：用于生成交易信号和策略判断的代码。
- trading_strategy: code used to generate trading signals and strategy decisions.

- risk_control：用于检查风险、限制交易和执行保护规则的代码。
- risk_control: code used to check risk, restrict trading, and enforce protection rules.

- trade_executor：用于向交易所发送订单和管理订单执行的代码。
- trade_executor: code used to send orders to exchanges and manage order execution.

- ai_agents：AI Agent 相关逻辑代码。
- ai_agents: code related to AI agent logic.

- daily_reports：用于生成日报、总结和复盘报告的代码。
- daily_reports: code used to generate daily reports, summaries, and review outputs.

- dashboard：用于可视化界面、本地仪表板或简单管理页面的代码。
- dashboard: code used for visual dashboards, local interfaces, or simple management pages.

这个目录只放代码，不放大型数据文件、数据库导出文件或日志文件。
This folder should contain code only, not large data files, database dumps, or log files.

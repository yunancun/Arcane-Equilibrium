## 6 大測試類型
1. **Unit test**：邊界值 + 正常路徑 + 安全攻擊路徑（修復後）
2. **Integration test**：跨模塊調用鏈
3. **Property-based (proptest)**：Rust 狀態機窮舉 + serde round-trip
4. **Concurrency test**：asyncio 多 task / 兩 worker race / shared singleton
5. **SLA 壓測**：閾值以 performance-profiling skill（唯一正本）為準
6. **Cross-language consistency**：Python ↔ Rust 1e-4 容差

## Mock 安全規則
- ✅ Mock：外部 IO（HTTP / PG / file system）/ 時間 / 隨機
- ❌ Mock：業務邏輯 / 計算函數 / IPC protocol 邏輯（會掩蓋真實 bug）

# local_ai_services

这里存放本地 AI 服务的部署文件。
This folder stores deployment files for local AI services.

这里主要预留给未来可能部署在本机上的 AI 相关服务，例如本地 embedding 模型、本地 reranker、本地推理服务或 Ollama 类服务。
This folder is mainly reserved for future AI-related services running on the local machine, such as local embedding models, local rerankers, local inference services, or Ollama-style services.

当前阶段系统主要仍以云端模型为主，但这里为未来充分利用本机算力预留扩展空间。
At the current stage, the system mainly relies on cloud models, but this folder reserves expansion space for future local AI usage.

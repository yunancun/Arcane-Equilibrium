## F2：部署過濾規則

✅ **代碼確認正確**

```python
# strategy_auto_deployer.py:191-197 _deploy_strategy()
if abs(getattr(opp, "price_change_pct_24h", 0.0)) > 40.0:
    logger.info("Skipping trend deploy for %s: extreme daily change=%.1f%%", ...)
    return

# market_scanner.py:149-154 符號基本過濾
if price < 0.01: continue     # 超低價過濾

```

機會評分規則（market_scanner.py）：
- Funding Arb：`min(100, funding_abs_bps × 3)`，$50M+ 成交量 1.2× 加成
- Grid Trading：`40 + volatility_pct × 3`（ranging + 2% < vol < 10%）
- Trend Following：`min(100, 30 + abs(price_change_pct) × 5)`
- Mean Reversion：`25 + (4 - volatility_pct) × 5`
- 最低門檻：score > 20

**Unknown regime 禁止：** MACrossoverStrategy 入場前檢查（詳見 P0-A1）

## C1：P0 硬停損觸發條件

⚠️ **P1 問題：session drawdown gate 無強制執行**

```python
# risk_manager.py:977-985
peak = sess.get("peak_balance_usdt", sess.get("initial_paper_balance_usdt", 0))
current = sess.get("current_paper_balance_usdt", 0)

    drawdown_pct = ((peak - current) / peak) * 100
    if drawdown_pct >= self._config.max_session_drawdown_pct:   # 15%
        if not sess.get("session_halted"):

                "Session drawdown %.1f%% >= %.1f%% → halt recommended",
```

- `max_session_drawdown_pct = 15.0`（GlobalRiskConfig line 282）

- **結論：** Session drawdown circuit breaker 只是記錄，無法真正停止交易（P1）

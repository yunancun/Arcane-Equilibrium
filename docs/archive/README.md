# Archive

> **HISTORICAL / COMPLETED / STALE**
>
> 本目录保存已完成、已归档、已 superseded 或从 active docs 中抽出的历史材料。
> 它不是 backlog，也不是当前 work queue。当前工作一律先读根目录 `TODO.md`。

## 何时读取

| 目的 | 入口 |
|---|---|
| 查某次 TODO / CLAUDE / README 清理前快照 | 顶层 `YYYY-MM-DD--*.md` 快照文件 |
| 查已完成 wave / sprint 叙事 | 顶层 archive 文件或对应子目录 `_README.md` |
| 查被替代的 REF/worklog 文档 | `2026-05-28--*_superseded/` 或 `2026-05-28--worklog_*_archived/` |
| 查当前主题证据入口 | 先读 `docs/_indexes/initiative_index.md`，再按链接进 archive |

## 不要误读

- Archive 中的 “DONE / active / ready” 只代表当时状态。
- Archive 中的 paper/demo/live 旧语义不得覆盖 `CLAUDE.md` hard boundaries、
  `.codex/MEMORY.md` 和当前 `TODO.md`。
- 除非有 redirect plan 和 PM approval，不要移动或删除 archive 证据。

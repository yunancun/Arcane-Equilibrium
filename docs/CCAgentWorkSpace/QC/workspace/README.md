# QC Workspace — 量化顧問工作區

## 目錄結構

```
workspace/
└── reports/          # 策略評審、風控提案、回測方法論等輸出文件
    └── YYYY-MM-DD--描述.md
```

## 報告類型

| 類型 | 命名模式 | 說明 |
|------|---------|------|
| 策略評審 | `YYYY-MM-DD--strategy_review_XXX.md` | 單策略或全策略組合的數學評審 |
| 風控提案 | `YYYY-MM-DD--risk_model_proposal_XXX.md` | 統計風控模型設計提案 |
| 回測方法論 | `YYYY-MM-DD--backtest_methodology_XXX.md` | 回測框架設計與驗證標準 |
| Alpha 研究 | `YYYY-MM-DD--alpha_research_XXX.md` | 新 edge 來源研究筆記 |
| 季度復盤 | `YYYY-MM-DD--quarterly_review_QXYYYY.md` | 策略組合定期健康檢查 |

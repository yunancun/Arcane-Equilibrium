# QC Design - ALR P2-4 Scanner Statistical Experiment

Date: 2026-07-09
Verdict: `ACCEPT_STATISTICAL_EXPERIMENT_ONLY`
Mode: `ROLE_FALLBACK_SINGLE_SESSION`

Scanner snapshots may rank the information value of a symbol, but contain no
candidate-matched realized after-cost return. The P2-4 experiment therefore
measures only deterministic scanner recurrence and novelty: occurrence count,
new-add count, and a same-cycle matched control set. It must record zero for
after-cost statistics rather than infer a return from scanner score/rank.

The selected target is a research question, not a trading candidate. Every
candidate artifact must state `DEFER_EVIDENCE` for candidate-matched fills,
fees, slippage, funding, reconstruction, controls, and repeat/OOS outcome
evidence. No score is an edge or profit claim.

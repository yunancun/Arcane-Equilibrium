## 5. 回測驗證要求(protocol 本體;執行=MIT/E4,QC 出帶裁決)

### 5.1 資料品質前置檢定(step 1,記錄性)
對 pooled 日報酬面板與每候選 cell 淨值序列跑 ADF/KPSS/Ljung-Box/Jarque-Bera/ARCH-LM;JB 必拒 normality、ARCH 顯著 → PSR skew/kurt 修正與 block bootstrap 為強制。異常 → 停下查資料。

### 5.2 Leak-free 強制
全特徵 shift(1);z-score 只用 rolling(90d)/expanding,禁全期 mean/std。signal spec 層寫斷言測試(point-in-time invariance:t 後資料置 NaN 不改 t 值),CI 級強制,含 z90/β/M4 權重全覆蓋。Purge+embargo:WF-fitted 元件 train/test 邊界 purge h 日 + embargo h 日(MIT 主審)。無 ML 訓練;後續加 ML 腿另立 prereg。

### 5.3 Walk-forward(rolling 90/30)
Rolling 90d train / 30d test。WF-fitted 元件僅三項:M4 IR 權重、shrunk-β、z90 統計。**Evidence series E26 = 全部 test block 串接的日淨報酬(taker+funding,demeaned);in-sample 不進任何判定。**

### 5.4 檢定
主檢定(信號層)daily Spearman IC(重疊窗 Newey-West lag=h / h-block;day-cluster G=交易日數);組合層 E26 日均淨 spread HAC t(lag=h);PSR(0) 含 skew/kurt 修正;DSR exploratory K=114 / primary K=1;年化 ×365。

### 5.5 Block bootstrap CI
Stationary bootstrap(Politis-Romano),block 長 2h 日,B=1,000 → SR_ann 95% CI、maxDD 95% CI;IID bootstrap 禁用。

### 5.6 PBO / CSCV
K=114≥10 → 強制;CSCV S=16;**PBO<0.5** 為 GO 必要條件。

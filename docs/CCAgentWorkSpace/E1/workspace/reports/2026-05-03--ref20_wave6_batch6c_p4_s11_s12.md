V041 stub 只 expose 4 column（experiment_id / half_life_days / embargo_days / created_at），未含 dispatch SQL 用的 expires_at / status。

**Forward-compat 自動升級**：當 P2b fixture deploy 後（V045 schema + replay.experiments 完整 column），caps 自動升級 6/6 True，filter 自動切 full path（caller 不需 code change）。

### 5.3 'mlde_advisor' alias 是否該保留

helper 把 'mlde_advisor' 列入 allowlist（dispatch 明列）。但 V036 注釋說 "'mlde_advisor' alias mapped to 'ml_shadow' upstream by Python producer; verified function 不接受 alias"。

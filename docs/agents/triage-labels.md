# Triage Labels

The mattpocock engineering skills use five canonical triage roles. This file maps those roles to the actual label strings used in this repo's GitHub issue tracker.

| Label in mattpocock skills | Label in GitHub Issues | Meaning |
|---|---|---|
| `needs-triage` | `needs-triage` | Maintainer needs to evaluate this issue |
| `needs-info` | `needs-info` | Waiting on reporter for more information |
| `ready-for-agent` | `ready-for-agent` | Fully specified, ready for an AFK agent |
| `ready-for-human` | `ready-for-human` | Requires human implementation |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill mentions a role, use the corresponding GitHub label string from this table.

If the GitHub repo does not already have one of these labels, create it deliberately or report the missing label instead of inventing a near-duplicate.

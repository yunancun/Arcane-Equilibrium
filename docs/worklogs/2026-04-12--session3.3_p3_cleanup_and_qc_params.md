## 技術決策 / Technical Decisions

1. **FIX-43 統一啟動功能移除**：tab-trading.html 的 `unifiedStart()` 自動讀取 Demo 餘額作為 Paper 初始值。移除後 tab-paper.html 有手動 Balance 輸入框，tab-demo.html 有獨立 session 控制。功能完整，操作多一步但結構清晰。

2. **BB Breakout 出場偏移固定**：出場信心 = `exit_conf_base + offset`（trailing +0.2 / regime +0.1 / pctb +0.05 / squeeze −0.05）。偏移表達相對優先級而非絕對值，不單獨參數化。Agent 調 `exit_conf_base` 整體偏移即可。

3. **Grid consts 保留為默認值**：`ADAPTIVE_RANGE_PCT` / `REJECT_BACKOFF_MS` consts 保留在文件中，僅作為 `new()` 構造函數的默認初始化值。運行時 on_tick 全部改用 `self.field`。Factory `create_with_params()` 從 TOML 覆蓋。

4. **OU update interval 守衛**：`self.ou_update_interval > 0` 防止配置為 0 時的除零 panic。設為 0 時等效於禁用 OU 重算。

5. **Hurst 門檻函數參數注入**：`hurst()` 簽名加 `trending_threshold` / `mean_reverting_threshold`，`compute_all()` 傳默認值 0.60/0.40。公開常量 `DEFAULT_HURST_TRENDING_THRESHOLD` / `DEFAULT_HURST_MEAN_REVERTING_THRESHOLD` 供未來 TOML 配線。不擴展 `compute_all()` 簽名 — 門檻調整場景罕見，需要時直接調用 `hurst()` 即可。

五、当前阶段的核心语义（必须接受，不要回退）
======================================================================

语义 A：合法 no-AI-call 是有效终态
如果同时满足：
- route_plan = route_skip
- selected_ai_tier = none
- should_call_ai = false

那么这表示“这一轮设计上本来就不该调用 AI”，属于合法 observation terminal state。
不能再因为：
- invocation_attempted = false
- provider_response_present = false
- parsed_json_present = false
就把整链判失败。

语义 B：合法 idle account 是 info/idle，不是 blocker
如果同时满足：
- read_only
- execution disabled
- 无持仓
- 无挂单
- 无执行历史

那么：
- no_open_positions
- no recent order history
- no recent execution history
- private ws control-only traffic
这些最多只能作为 info / idle_flags。

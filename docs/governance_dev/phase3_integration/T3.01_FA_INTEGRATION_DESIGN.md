## 1. Integration Strategy / 集成策略

### 核心原則
- **非侵入式集成**：不修改現有 SM 模組的內部邏輯，僅在邊界處注入
- **單例模式**：每個 SM 全局唯一實例，在 `paper_trading_routes.py` 中初始化
- **Fail-closed**：SM 不可用時，系統拒絕所有需要授權的操作
- **漸進啟用**：可通過配置開關逐一啟用 SM，不需全部同時上線

## 2. File-by-File Design / 逐文件設計

### 2.1 `governance_hub.py` (NEW — 核心集成層)

```python
"""
Governance Hub — Central integration point for all governance state machines.
治理集線器 — 所有治理狀態機的中央集成點。
"""

class GovernanceHub:
    """
    Orchestrates AuthorizationSM, RiskGovernor, DecisionLease, ReconciliationEngine.

    Wiring rules:
    - RiskGovernor.on_escalation → AuthSM.restrict() or .freeze()
    - ReconciliationEngine.on_mismatch → RiskGovernor.escalate()
    - AuthSM.on_frozen → LeaseSM.revoke_all_active()
    - All transitions → AuditPersistence
    """

    def __init__(self, *, audit_dir: str, enabled: bool = True):
        self.auth_sm = AuthorizationStateMachine(...)
        self.risk_gov = RiskGovernor(...)
        self.lease_sm = DecisionLeaseStateMachine(...)
        self.recon = ReconciliationEngine(...)
        self._enabled = enabled

        # Wire cross-SM callbacks
        self.risk_gov.on_escalation = self._on_risk_escalation
        self.recon.on_mismatch = self._on_reconciliation_mismatch

    def is_authorized(self, scope: str) -> bool:
        """H0 gate: check if current authorization allows operation."""
        if not self._enabled:
            return False  # fail-closed
        return self.auth_sm.is_authorized()

    def check_risk_and_restrict(self, metrics: dict) -> RiskLevel:
        """Feed risk metrics, auto-escalate if needed, restrict auth if necessary."""
        ...

    def acquire_lease(self, intent: dict) -> str | None:
        """Acquire a decision lease for an order intent. Returns lease_id or None."""
        ...

    def reconcile(self, paper_state: dict, demo_state: dict) -> dict:
        """Run reconciliation between paper and demo states."""
        ...

    def get_status(self) -> dict:
        """Return combined governance status for API/GUI."""
        ...
```

### 2.2 `governance_routes.py` (NEW — REST API 端點)

```
6 routes under /api/v1/governance/:
  GET  /status          — Combined governance dashboard
  GET  /auth/status     — Authorization SM state
  POST /auth/approve    — Operator approves pending authorization
  GET  /risk/level      — Current risk governor level
  POST /risk/override   — Operator de-escalates risk (requires approval)
  POST /reconcile/run   — Trigger manual reconciliation
```

### 2.3 `paper_trading_routes.py` (MODIFY — 注入 GovernanceHub)

Changes:
- Import and instantiate GovernanceHub
- Pass hub to ENGINE constructor: `ENGINE = PaperTradingEngine(PAPER_STORE, risk_manager=RISK_MANAGER, governance_hub=GOV_HUB)`
- Add `is_authorized()` check before session start
- Add `reconcile()` after session stop

### 2.4 `paper_trading_engine.py` (MODIFY — 接收 GovernanceHub)

Changes:
- `__init__` accepts optional `governance_hub` parameter
- `submit_order()`: if hub exists, check `hub.is_authorized()` and `hub.acquire_lease()` before processing
- `_fill_order()`: feed fill result to risk governor metrics
- `stop_session()`: trigger reconciliation

### 2.5 `pipeline_bridge.py` (MODIFY — Lease 集成)

Changes:
- `process_pending_intents()`: before submitting intent to engine, call `hub.acquire_lease(intent)`
- If lease denied, reject intent with reason

### 2.6 `risk_manager.py` (MODIFY — Risk Governor 聯動)

Changes:
- After computing `risk_pressure`, feed to `hub.risk_gov.evaluate_metrics()`
- Governor level affects `_effective_max_position_pct()` scaling

## 3. Cross-SM Wiring / 跨狀態機聯動

| Trigger | Source SM | Target SM | Action |
|---------|-----------|-----------|--------|
| Risk ≥ REDUCED | RiskGovernor | AuthSM | `restrict()` |
| Risk ≥ CIRCUIT_BREAKER | RiskGovernor | AuthSM | `freeze()` |
| Risk ≥ DEFENSIVE | RiskGovernor | LeaseSM | `revoke_all_active()` |
| Auth → FROZEN | AuthSM | LeaseSM | `revoke_all_active()` |
| Recon MISMATCH_MAJOR | Recon | RiskGovernor | `escalate(CRITICAL)` |
| Recon FATAL | Recon | AuthSM | `freeze()` |
| Lease EXPIRED (bulk) | LeaseSM | AuditLog | `log_bulk_expiry()` |

## 5. Enable/Disable Strategy / 啟用/停用策略

Environment variable: `OPENCLAW_GOVERNANCE_ENABLED=true|false` (default: `true`)

When disabled:
- GovernanceHub returns `is_authorized() = False` (fail-closed, not bypass!)
- To run in legacy mode: `OPENCLAW_GOVERNANCE_MODE=legacy` skips hub entirely

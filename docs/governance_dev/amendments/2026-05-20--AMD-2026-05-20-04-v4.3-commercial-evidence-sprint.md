## 3. 4 條 Push Back 詳細理由

### 3.1 Subscription cost cycling — sunk cost framing

Reviewer 把 $220-300 / $100-150 / $40-120 寫成 dollar cap，假設 operator
能根據 dev intensity cycle Claude Max + ChatGPT Plus 訂閱。

**反駁事實**：
- 月度訂閱無 prorated cancel（取消即失效，重啟需重訂）
- Operator 跨多個 personal + professional 活動共用兩個訂閱（不只此項目）
- 訂閱是 operator 整體 life expense 的一部分，不是此項目 attributable cost
- "Low Activity Mode $40-120" 假設 operator 完全取消兩個訂閱 — 不現實

**v4.3 修正**：ADR-0027 Plan Mode 控制 operator weekly hours，不控制 dollar cycling。

```
v4.3 真實 burn:
  Fixed (sunk): Claude Max $200/mo + ChatGPT Plus $200/mo
                 = $400/mo, NOT this-project attributable
  Variable: API spend (cap by mode), hosting $20-30, operator time opp cost
  This-project marginal burn / yr: $400-500 (API + hosting)
  
v4.3 marginal-burn 12-month projection: $400-500
v4.3 EV (probability-weighted, v4.3 §12): $488-2100
Net EV: ≈ break-even to +$1700
```

更誠實的版本。

### 3.2 8-12 週 timebox arbitrary

Reviewer 給 8-12 週是日曆 number。但 LCS evidence 真實 maturity timeline：

```
W1-W2: market.liquidations 從 2.12d 累積到 ~16d sample
W3-W4: event-study window (n_events ~ 50-100, t-stat possible)
W5-W6: 14d demo deploy start
W7-W8: 14d demo cumulative evidence
W9-W12: 30d 二次確認窗口（若 W8 marginal）
```

→ 8 週 = 最早 evidence point；12 週 = 確認 window。**reviewer 給對日期但理由
錯**。v4.3 §0 + sprint plan 明確把這條綁在 evidence maturity 而非日曆。

### 3.3 Monetization demand test 需真實 spec

reviewer 寫 "Copy Trading / Signal waitlist / buyer interest" 但**未指定**:
- 平台選擇（什麼工具）
- 量化標準（什麼算「demand」）
- 法律框架（ToS / disclaimer）
- 時間/工程成本估計

v4.3 補完整 spec：`docs/execution_plan/2026-05-20--monetization-demand-test-spec.md`
(~410 LOC + 23 hr setup + 5-level demand pyramid + L4 paid pre-order threshold)。

### 3.4 IP sale prep — W1 漸進非 W12 fallback

Reviewer 把 IP sale 列為 W12+ kill fallback。但：

- IP sale prep 工程量低（~16-24 hr 跨 8 週）
- 不和 Stream 1 / 2 衝突資源
- 即使 Stream 1 + 2 全 success，IP value 仍 > $0
- 延到 W12 才開始 = 浪費 8 週 option value time

→ v4.3 Stream 3 W1 開始漸進 prep（README cleanup + diagram + 5-min demo
video + private landing page + quiet outreach 5-10 個 networks）。

ROI 計算（v4.3 §3）：
- Cost: 16-24 hr operator
- EV (probability-weighted): $1.5-4.5k
- ROI/hr: $62-280

顯著高於 Track A live $1k expected annual ROI。

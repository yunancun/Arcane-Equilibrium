# stored_data

这里存放交易 AI 系统运行过程中产生和使用的主要数据。
This folder stores the main data generated and used by the trading AI system.

它包括行情数据、新闻与情报、交易信号、回测结果、模拟交易数据、实盘交易数据、资金追踪数据以及 AI 模型文件。
It includes market data, news and intelligence, trading signals, backtest results, simulated trading data, live trading data, fund tracking data, and AI model files.

主要子目录说明：
Main subfolder overview:

- market_data：原始行情数据、清洗后的数据、特征数据和市场快照。
- market_data: raw market data, cleaned data, calculated features, and market snapshots.

- news_and_intelligence：新闻原文、解析后的新闻和向量记忆数据。
- news_and_intelligence: raw news, parsed news, and vector memory data.

- trading_signals：策略或 AI Agent 生成的交易信号。
- trading_signals: trading signals generated by strategies or AI agents.

- backtest_results：回测结果和研究输出。
- backtest_results: backtest results and research outputs.

- demo_trading：交易所官方 Demo 环境中的交易数据。
- demo_trading: trading data from exchange-provided demo environments.

- paper_trading：系统内部模拟交易数据。
- paper_trading: internally simulated trading data.

- live_trading：真实实盘交易数据。
- live_trading: real live trading data.

- fund_tracking：资金流、稳定币、平台手续费币、费用、PnL 和划转数据。
- fund_tracking: cash flow, stablecoins, platform fee tokens, costs, PnL, and transfer data.

- ai_models：本地模型文件或 AI 相关模型资产。
- ai_models: local model files or AI-related model assets.

本目录的核心组织逻辑是：按交易环境、产品类型、交易方式和交易记录来管理数据。
The core organizing logic of this folder is based on trading environment, product type, trading method, and trading records.

币种和交易对不应当被建立成大量文件夹，而应当出现在文件名、数据库字段、元数据或配置文件中。
Coins and trading pairs should not be turned into large numbers of folders, but should appear in file names, database fields, metadata, or configuration files.

# backup_files

这里存放本地备份、备份记录、NAS 同步记录和归档数据。
This folder stores local backups, backup records, NAS sync records, and archived data.

主要子目录包括：
Main subfolders include:

- local_backups：本机临时备份文件。
- local_backups: temporary local backup files.

- backup_records：备份记录和备份清单。
- backup_records: backup records and backup manifests.

- nas_sync_records：和 NAS 同步相关的记录。
- nas_sync_records: records related to NAS synchronization.

- database_backups：数据库导出备份。
- database_backups: exported database backups.

- data_archives：长期归档数据。
- data_archives: long-term archived data.

这个目录主要服务于系统恢复、灾难恢复和长期归档管理。
This folder mainly serves system recovery, disaster recovery, and long-term archive management.
